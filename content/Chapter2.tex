    \section{Listings}
        Example file listing seen in \autoref{lst:bsp}:
        \begin{listing}[H]
            \inputminted[breaklines, fontsize=\small, linenos, numbersep=5pt]{cpp}{media/Skripte/example.cpp}
            \caption{Example Listing}
            \label{lst:bsp}
        \end{listing}
        Or inline (\autoref{lst:bsp_inline}):
        \begin{listing}[H]
            \begin{minted}[breaklines, fontsize=\small, linenos, numbersep=5pt, autogobble]{bash}
                echo "example" >> /dev/null
            \end{minted}
            \caption{Inline Listing}
            \label{lst:bsp_inline}
        \end{listing}
        Make sure to activate shell-escape to use minted!
